## Simulation

The Rust simulation crate for the hybrid NTP study lives in `crates/ntp-hybrids-sim`.

- CLI crate: `crates/ntp-hybrids-sim`
- Colab notebook: `crates/ntp-hybrids-sim/ntp-hybrids-sim-visualization.ipynb`
- Output root: `output-ntp-hybrids-sim`

The notebook button above opens the visualization workflow directly in Google Colab so the crate can be built, run, and plotted without a local Rust setup.
